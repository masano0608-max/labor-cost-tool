# GitHub トークンの作り方

`git push` でパスワードを聞かれたら、**Personal Access Token（パーソナルアクセストークン）** を作って、それを「パスワード」の欄に入力します。

---

## やり方（5ステップ）

### ステップ1：GitHub にログイン

1. ブラウザで https://github.com を開く
2. ログインする

---

### ステップ2：トークン作成ページを開く

1. 画面右上の **自分のアイコン**（丸い写真）をクリック
2. **Settings**（設定）をクリック
3. 左メニューの一番下までスクロール
4. **Developer settings**（開発者設定）をクリック
5. 左メニューの **Personal access tokens** → **Tokens (classic)** をクリック
6. **Generate new token** → **Generate new token (classic)** をクリック

※ パスワードを聞かれたら、GitHub のログイン用パスワードを入力

---

### ステップ3：トークンを作る

1. **Note**（メモ）の欄に、何でもいいので名前を入れる  
   例：`パソコン用` や `push用`

2. **Expiration**（有効期限）で **90 days** か **No expiration** を選ぶ

3. **Select scopes** で **repo** にチェックを入れる  
   （repo を1つ入れると、その下の項目も自動で選ばれます）

4. 一番下の **Generate token**（緑のボタン）をクリック

---

### ステップ4：トークンをコピーする

- **ghp_xxxxxxxxxxxx** のような文字列が表示されます
- これを **必ずコピー** してください
- この画面を閉じると、もう二度と表示されません

---

### ステップ5：ターミナルで使う

1. ターミナルに戻る

2. **Username** の欄に GitHub のユーザー名を入力して Enter  
   （例：`masano0608-max`）

3. **Password** の欄に、今コピーしたトークン（`ghp_...`）を貼り付けて Enter  
   ※ 貼り付けたときは画面に何も表示されませんが、問題ありません

4. これでプッシュが実行されます

---

## もっと簡単な方法：GitHub のサイトからアップロード

トークンが難しい場合は、**ブラウザから直接ファイルをアップロード**する方法もあります。
