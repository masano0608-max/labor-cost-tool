# 最終デプロイ手順

あなたがやることは **2つだけ** です。

---

## やること1：GitHub に SSH 鍵を登録する

SSH 鍵（パソコンの鍵）はもう作ってあります。  
これを GitHub に登録するだけで、パスワードなしで push できるようになります。

### 手順

1. ブラウザで次のアドレスを開く  
   **https://github.com/settings/ssh/new**  
   （ログインしていなければ、先にログインしてください）

2. **Title** の欄に名前を入れる（何でもOK）  
   例：`MacBook`

3. **Key** の欄に **⌘+V で貼り付ける**  
   （すでにクリップボードにコピー済みです）  
   `ssh-ed25519 AAAAC3Nza...` のような長い文字列が貼り付けられます

4. **Add SSH Key**（緑のボタン）をクリック

5. パスワードを聞かれたら、GitHub のパスワードを入力

**これで登録完了です。**

---

## やること2：「できたよ」と教えてください

SSH 鍵の登録が終わったら、このチャットに「できた」と送ってください。  
残りの操作（push・デプロイ）はこちらで自動実行します。

---

## 補足

- SSH 鍵は1回登録すれば、今後ずっと使えます
- パスワードやトークンを毎回入力する必要はなくなります
- Key の欄が空のときは、ターミナルで次を実行してからもう一度 ⌘+V してください：

```
cat /Users/masanotanaka/.ssh/id_ed25519.pub | pbcopy
```
