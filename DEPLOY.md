# サイトをインターネットに公開する方法

「このツールを、他の人も使えるURLで公開したい」ときの手順です。

---

## 一番簡単な方法：Netlify（パソコンに何も入れない）

**ターミナルもコマンドも不要です。**

### 手順

1. **https://app.netlify.com/drop** をブラウザで開く

2. **このフォルダ（2026.2.26）を、そのままドラッグして画面にドロップする**
   - フォルダの中身（index.html など）ではなく、**フォルダごと**ドロップ

3. 少し待つと「Your site is live」と出る

4. 表示されたURL（例：`https://xxx.netlify.app`）をクリックすると、サイトが開く

**以上です。** これだけで公開できます。

---

## Google のサーバーで公開する方法：Firebase

「Google のサービスで公開したい」場合の手順です。

---

### 準備：必要なもの

- Google アカウント（Gmail のアカウント）
- Node.js（入ってない場合は https://nodejs.org から「LTS」をダウンロードしてインストール）

---

### ① Google のサイトで「プロジェクト」を作る

1. ブラウザで **https://console.firebase.google.com/** を開く

2. **「プロジェクトを追加」** というボタンをクリック

3. 「プロジェクト名」に好きな名前を入れる（例：`人件費ツール`）

4. 「続行」→「続行」と進めて、最後に「プロジェクトを作成」をクリック

5. 作成できたら、画面左上の **⚙️（歯車）** をクリック

6. **「プロジェクトの設定」** をクリック

7. 画面の「プロジェクト ID」のところに、英数字の文字列が出ている（例：`labor-cost-abc123`）
   - **これをメモ帳にコピーして貼り付けておく**
   - あとで使います

---

### ② ターミナルを開く

**Mac の場合：**
- 「Spotlight」で「ターミナル」と検索して開く
- または「アプリケーション」→「ユーティリティ」→「ターミナル」

**Windows の場合：**
- 「コマンドプロンプト」または「PowerShell」を開く
- スタートメニューで「cmd」と検索すると出てきます

---

### ③ このフォルダに移動する

ターミナルに、次を入力して Enter：

```
cd 2026.2.26
```

※ フォルダの場所が違う場合は、`cd` のあとにフォルダのパスを入れます。
例：`cd /Users/あなたの名前/2026.2.26`

---

### ④ Firebase のツールを入れる

ターミナルに、次を入力して Enter：

```
npm install -g firebase-tools
```

「インストールしています…」のように出たら、終わるまで待ちます。

---

### ⑤ Google にログインする

ターミナルに、次を入力して Enter：

```
firebase login
```

- ブラウザが開くので、Google アカウントでログイン
- 「Firebase CLI にアクセスを許可しますか？」→ **許可** をクリック
- ターミナルに「Success」と出たらOK

---

### ⑥ プロジェクトIDを設定する

1. このフォルダにある **`.firebaserc`** というファイルを開く  
   （テキストエディタやメモ帳でOK）

2. 中身がこんな感じになっている：
   ```
   "default": "YOUR_PROJECT_ID"
   ```

3. **`YOUR_PROJECT_ID`** の部分を、①でメモした「プロジェクト ID」に置き換える

   **例：**
   - メモしたのが `labor-cost-abc123` なら
   - `"default": "labor-cost-abc123"` に書き換える

4. ファイルを保存する

---

### ⑦ デプロイする（アップロードする）

ターミナルに、次を入力して Enter：

```
firebase deploy --only hosting
```

- 「Deploy complete!」と出たら成功
- 「Hosting URL: https://〇〇〇.web.app」のような行が出る
- そのURLをブラウザで開くと、サイトが表示されます

---

## 困ったとき

| こんな表示が出た | やること |
|------------------|----------|
| `firebase が見つかりません` | Node.js をインストールしてから、④をもう一度やる |
| `Failed to get Firebase project` | ⑥の「プロジェクトID」が間違っていないか確認する |
| `Permission denied` | ⑤の `firebase login` をもう一度やる |

---

## どっちを選べばいい？

- **とにかく簡単にやりたい** → 一番上の「Netlify」の方法
- **Google のサーバーで公開したい** → Firebase の方法
