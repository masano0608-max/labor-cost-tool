# Settings → Pages の画面での次のステップ

「Source: GitHub Actions」が選ばれていれば、**この画面での設定はもう終わりです。**

---

## やらなくていいこと

- **Next.js、NuxtJS、Static HTML などの「Configure」ボタンは押さない**
- このプロジェクトにはすでにワークフローが入っているので、不要です

---

## 次にやること

### ① Actions タブを開く

1. リポジトリのページの**一番上**に戻る（「Settings」の左にある「Code」をクリック）
2. 上のタブの中から **「Actions」** をクリック

### ② ワークフローが動いているか確認する

- 「Deploy to GitHub Pages」というワークフローが表示される
- 左側に **黄色い丸（実行中）** や **緑のチェック✓（完了）** が出る
- 初回は 1〜2 分ほどかかることがあります

### ③ 緑の✓ が出たら

1. もう一度 **Settings** → **Pages** を開く
2. ページの上の方に **「Your site is live at ○○○」** と表示される
3. その URL をクリックすると、サイトが開きます

---

## まとめ

| 今の画面で | やること |
|-----------|----------|
| Source が GitHub Actions | 何も押さず、そのまま |
| Configure ボタン | **押さない** |
| 次に | **Actions** タブで完了を待つ |
